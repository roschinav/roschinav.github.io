Загрузочная флэшка DOS

Для создания загрузочной флэшки надо восстановить ее образ (см.ссылку) на флэшке, размером не менее 256 МБ. Образ создан программой Acronis True Image 2014. Восстанавливать надо той же программой.

Если программы Acronis True Image 2014 нет, можно создать загрузочную флэшку так, как это описаено в: https://remontka.pro/dos-boot-usb/. Скачать программу Rufus можно тут: https://rufus.ie/. Однако, после создания флэшки надо будет отредактировать файлы autoexec.bat и config.sys

Флэшка будет работать на любом х86 компьютере. Надо в Setup задать загрузку с Flash USB.

На компьютерах фирмы Apple флэшка загружаться не будет!!!
