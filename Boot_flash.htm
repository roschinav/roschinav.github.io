Загрузочная флэшка DOS

Для создания загрузочной флэшки надо восстановить ее образ (см.ссылку) на флэшке, размером не менее 256 МБ. Образ создан программой Acronis True Image 2014. Восстанавливать надо той же программой. Флэшка будет работать на любом х86 компьютере. Надо в Setup задать загрузку с Flash USB.

На компьютерах фирмы Apple флэшка загружаться не будет!!!
