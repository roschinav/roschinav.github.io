TASM

tasm.exe - компилятор с мнемоник ассемблера в объектный код. Если исходный файл (например, prog.asm) имеет расширение .asm, то при компиляции расширение файла указывать не обязательно (tasm prog). Если расширение другое (например, prog.qqq), расширение указывать обязательно (tasm prog.qqq).

Для получения справки по tasm, надо набрать: tasm /?

TLINK

tlink.exe - линковщик, который преобразует объектные коды в исполняемый файл. Если исходный файл (например, prog.obj) имеет расширение .obj, то при линковке расширение файла указывать не обязательно (tlink prog). Если расширение другое (например, prog.sss), расширение указывать обязательно (tlink prog.sss).

Если надо получить исполняемый файл типа .com, надо указать ключ /t (tlink prog /t). Внимание! Буква t в ключе должна быть строчной (маленькой). Если надо получить исполняемый файл типа .exe, ключ не нужен.

Для получения справки по tlink, надо набрать: tlink /?

NC - Нортон коммандер

Нортон коммандер - это командная оболочка, позволяющая удобно работать с файлами. Запускаемый файл nc.exe

Компилятор tasm.exe, линковщик tlink.exe, Нортон коммандер NC\nc.exe
